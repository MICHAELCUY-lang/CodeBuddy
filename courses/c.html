Lesson 1: Introduction to C Programming

üéØ What is the C Language?

The C language is the "God's Programming Language" created by Dennis Ritchie between 1969-1973. C is a programming language that is:

  • Very powerful and efficient
  • The basis for modern languages (C++, C#, Java, PHP)
  • Used to create operating systems (Linux, Unix, Windows)
  • Low-level yet portable

üìã Basic C Program Structure

#include <stdio.h> int main() { printf("Hello, World!\n"); return 0; }

Component Explanation:

  • #include <stdio.h> - Includes the standard input/output library
  • int main() - The main function of the program
  • printf() - Function for printing output
  • return 0; - Ends the program with a success status

üíª printf() - Displaying Output

printf("Welcome to C!\n"); // \n is the escape character for a new line // Other escape characters: // \t = tab // \\ = backslash // \" = quotation mark
üí° Tip: Every statement in C must end with a semicolon (;). This is crucial!

üìù Comments in C

// This is a single-line comment /* This is a multi-line comment that can span several lines */

Lesson 2: Variables & Data Types

üì¶ What is a Variable?

A variable is a location in memory where we store data. Each variable has:

  • Name (identifier)
  • Data type
  • Value

üî§ Basic Data Types in C

// Integer - whole numbers int age = 25; int temperature = -10; // Float - decimal numbers (6-7 digits precision) float height = 175.5; float weight = 65.3; // Double - decimal numbers (15-16 digits precision) double pi = 3.14159265359; // Character - single character char grade = 'A'; char letter = 'Z'; // String - collection of characters (array of char) char name[] = "John Doe";

üìä Data Type Sizes

  • int - 4 bytes (usually)
  • float - 4 bytes
  • double - 8 bytes
  • char - 1 byte

üîß Declaration and Initialization

// Declaration (just creating the variable) int number; // Initialization (giving an initial value) number = 10; // Declaration + Initialization simultaneously int value = 100; // Multiple declaration int a, b, c; a = 1; b = 2; c = 3; // Multiple declaration + initialization int x = 5, y = 10, z = 15;

üìù scanf() - Getting Input

#include <stdio.h> int main() { int age; float height; char name[50]; printf("Enter your name: "); scanf("%s", name); printf("Enter your age: "); scanf("%d", &age); printf("Enter your height (cm): "); scanf("%f", &height); printf("\nName: %s\n", name); printf("Age: %d years\n", age); printf("Height: %.1f cm\n", height); return 0; }

Format Specifiers:

  • %d - integer
  • %f - float/double
  • %c - character
  • %s - string
  • %lf - double (for scanf)
‚ö†Ô∏è Attention: Use the & (ampersand) operator before the variable name in `scanf`, except for strings!

Lesson 3: Operators & Expressions

‚ûï Arithmetic Operators

int a = 10, b = 3; // Addition int result = a + b; // 13 // Subtraction result = a - b; // 7 // Multiplication result = a * b; // 30 // Division (integer division) result = a / b; // 3 (not 3.333...) // Modulus (remainder) result = a % b; // 1
‚ö†Ô∏è Important: Integer division in C will discard the decimal part!
For a decimal result, use float or double.

üìä Assignment Operators

int x = 10; // Compound assignment x += 5; // x = x + 5 ‚Üí 15 x -= 3; // x = x - 3 ‚Üí 12 x *= 2; // x = x * 2 ‚Üí 24 x /= 4; // x = x / 4 ‚Üí 6 x %= 4; // x = x % 4 ‚Üí 2 // Increment & Decrement x++; // x = x + 1 ++x; // x = x + 1 x--; // x = x - 1 --x; // x = x - 1

üîç Comparison Operators

int a = 5, b = 10; // The result of a comparison operator is 0 (false) or 1 (true) a == b // 0 (equals?) a != b // 1 (not equals?) a > b // 0 (greater than?) a < b // 1 (less than?) a >= b // 0 (greater than or equal?) a <= b // 1 (less than or equal?)

üéØ Logical Operators

int x = 5, y = 10, z = 15; // AND (&&) - both conditions must be true (x < y && y < z) // 1 (true) // OR (||) - at least one condition is true (x > y || y < z) // 1 (true) // NOT (!) - inverts the condition !(x == y) // 1 (true)

üìê Full Program Example

Lesson 4: Control Structures

üîÄ IF Statement

int score = 85; if (score >= 80) { printf("Grade A - Excellent!\n"); } // IF-ELSE if (score >= 70) { printf("Pass\n"); } else { printf("Fail\n"); } // IF-ELSE IF-ELSE if (score >= 90) { printf("Grade: A\n"); } else if (score >= 80) { printf("Grade: B\n"); } else if (score >= 70) { printf("Grade: C\n"); } else { printf("Grade: D\n"); }

üéõÔ∏è SWITCH Statement

int day = 3; switch (day) { case 1: printf("Monday\n"); break; case 2: printf("Tuesday\n"); break; case 3: printf("Wednesday\n"); break; case 4: printf("Thursday\n"); break; case 5: printf("Friday\n"); break; default: printf("Weekend!\n"); }
‚ö†Ô∏è Don't forget BREAK! Without `break`, the program will continue executing the next `case`.

Lesson 5: Functions & Recursion

üéØ What is a Function?

A function is a reusable block of code to perform a specific task.

üìã Function Structure

// Syntax: // return_type function_name(parameters) { // // body // } // Function without parameters and return void greet() { printf("Hello, welcome!\n"); } // Function with parameters void greetName(char name[]) { printf("Hello, %s!\n", name); } // Function with return value int add(int a, int b) { return a + b; } // Function with multiple parameters float average(float a, float b, float c) { return (a + b + c) / 3.0; }

üîß Using Functions

#include <stdio.h> // Function prototype (declaration) int multiply(int x, int y); float divide(float x, float y); int main() { int result1 = multiply(5, 3); printf("5 x 3 = %d\n", result1); float result2 = divide(10.0, 3.0); printf("10 / 3 = %.2f\n", result2); return 0; } // Function definition (implementation) int multiply(int x, int y) { return x * y; } float divide(float x, float y) { if (y != 0) { return x / y; } else { printf("Error: Divisor cannot be 0!\n"); return 0; } }

üéÆ Recursion

Recursion is a technique where a function calls itself to solve a task. Recursion is very useful for problems that can be solved by breaking down a large problem into smaller ones.

Recursion Example: Factorial

#include <stdio.h> // Recursive function to calculate factorial int factorial(int n) { if (n == 0) { return 1; // Base case: 0! = 1 } else { return n * factorial(n - 1); // Recursive step: n! = n * (n-1)! } } int main() { int number = 5; printf("Factorial of %d is %d\n", number, factorial(number)); return 0; }
üí° Tip: Every recursion needs a base case to stop the repeated calls.

üîß Functions in a Complete C Program

#include <stdio.h> // Function declaration int add(int a, int b); int main() { int result = add(5, 10); printf("5 + 10 = %d\n", result); return 0; } // Function definition int add(int a, int b) { return a + b; }

Lesson 6: Arrays & Pointers

üìö What is an Array?

An array is a collection of variables of the same data type arranged in a specific order. Arrays are very useful for storing many values under a single variable name.

üìã Array Declaration and Access

#include <stdio.h> int main() { // Declare an array with 5 elements int numbers[5] = {10, 20, 30, 40, 50}; // Accessing array elements printf("First element: %d\n", numbers[0]); // 10 printf("Second element: %d\n", numbers[1]); // 20 return 0; }

üìä Multidimensional Arrays

#include <stdio.h> int main() { // Declare a 2D array (matrix) int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}}; // Accessing 2D array elements printf("Matrix[0][1] = %d\n", matrix[0][1]); // 2 printf("Matrix[1][2] = %d\n", matrix[1][2]); // 6 return 0; }

üîç What is a Pointer?

A pointer is a variable that stores the memory address of another variable. Pointers are used to access and manipulate data at a specific memory location.

üìã Pointer Declaration and Usage

#include <stdio.h> int main() { int number = 10; int *ptr = &number; // Pointer ptr stores the address of number // Accessing the value through the pointer printf("Value of number: %d\n", *ptr); // Output: 10 printf("Address of number: %p\n", ptr); // Output: memory address of number return 0; }

üìä Pointers on Arrays

#include <stdio.h> int main() { int numbers[3] = {10, 20, 30}; int *ptr = numbers; // Pointer to the array // Accessing array elements using a pointer printf("First element: %d\n", *(ptr + 0)); printf("Second element: %d\n", *(ptr + 1)); return 0; }

üí° Importance of Pointers

Pointers are crucial in dynamic memory management, performance optimization, and interacting with the operating system or hardware at a lower level.
