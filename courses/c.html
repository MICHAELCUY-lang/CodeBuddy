Pelajaran 1: Pengenalan C Programming

ğŸ¯ Apa itu Bahasa C?

Bahasa C adalah "God's Programming Language" yang dibuat oleh Dennis Ritchie pada tahun 1969-1973. C adalah bahasa pemrograman yang:

  • Sangat powerful dan efficient
  • Menjadi basis dari bahasa modern (C++, C#, Java, PHP)
  • Digunakan untuk membuat sistem operasi (Linux, Unix, Windows)
  • Low-level namun tetap portable

ğŸ“‹ Struktur Program C Dasar

Penjelasan Komponen:

  • #include <stdio.h> - Menyertakan library standar input/output
  • int main() - Fungsi utama program
  • printf() - Fungsi untuk mencetak output
  • return 0; - Mengakhiri program dengan status sukses

ğŸ’» printf() - Menampilkan Output

Pelajaran 2: Variabel & Tipe Data

ğŸ“¦ Apa itu Variabel?

Variabel adalah lokasi di memori tempat kita menyimpan data. Setiap variabel memiliki:

  • Nama (identifier)
  • Tipe data
  • Nilai

ğŸ”¤ Tipe Data Dasar di C

ğŸ“Š Ukuran Tipe Data

  • int - 4 bytes (biasanya)
  • float - 4 bytes
  • double - 8 bytes
  • char - 1 byte

Pelajaran 5: Function & Rekursi

ğŸ¯ Apa itu Function?

Function adalah blok kode yang dapat digunakan kembali untuk melakukan tugas tertentu.

ğŸ“‹ Struktur Function

// Syntax: // return_type function_name(parameters) { // // body // } // Function tanpa parameter dan return void salam() { printf("Halo, selamat datang!\n"); } // Function dengan parameter void sapaNama(char nama[]) { printf("Halo, %s!\n", nama); } // Function dengan return value int tambah(int a, int b) { return a + b; } // Function dengan multiple parameters float rataRata(float a, float b, float c) { return (a + b + c) / 3.0; }

ğŸ”§ Menggunakan Function

#include <stdio.h> // Function prototype (deklarasi) int kali(int x, int y); float bagi(float x, float y); int main() { int hasil1 = kali(5, 3); printf("5 x 3 = %d\n", hasil1); float hasil2 = bagi(10.0, 3.0); printf("10 / 3 = %.2f\n", hasil2); return 0; } // Function definition (implementasi) int kali(int x, int y) { return x * y; } float bagi(float x, float y) { if (y != 0) { return x / y; } else { printf("Error: Pembagi tidak boleh 0!\n"); return 0; } }

ğŸ® Rekursi

Rekursi adalah teknik di mana sebuah function memanggil dirinya sendiri untuk menyelesaikan tugas. Rekursi sangat berguna untuk masalah yang dapat dipecahkan dengan membagi masalah besar menjadi masalah yang lebih kecil.

Contoh Rekursi: Faktorial

#include <stdio.h> // Fungsi rekursif untuk menghitung faktorial int faktorial(int n) { if (n == 0) { return 1; // Basis: 0! = 1 } else { return n * faktorial(n - 1); // Rekursif: n! = n * (n-1)! } } int main() { int angka = 5; printf("Faktorial dari %d adalah %d\n", angka, faktorial(angka)); return 0; }
ğŸ’¡ Tips: Setiap rekursi membutuhkan kondisi dasar (base case) untuk menghentikan pemanggilan berulang.

ğŸ”§ Fungsi dalam Program C Lengkap

#include <stdio.h> // Deklarasi fungsi int tambah(int a, int b); int main() { int hasil = tambah(5, 10); printf("5 + 10 = %d\n", hasil); return 0; } // Definisi fungsi int tambah(int a, int b) { return a + b; }

Pelajaran 6: Array & Pointer

ğŸ“š Apa itu Array?

Array adalah kumpulan variabel yang memiliki tipe data yang sama dan disusun dalam urutan tertentu. Array sangat berguna untuk menyimpan banyak nilai dalam satu nama variabel.

ğŸ“‹ Deklarasi dan Akses Array

ğŸ“Š Array Multidimensi

ğŸ” Apa itu Pointer?

Pointer adalah variabel yang menyimpan alamat memori dari variabel lain. Pointer digunakan untuk mengakses dan memanipulasi data di lokasi memori tertentu.

ğŸ“‹ Deklarasi dan Penggunaan Pointer

ğŸ“Š Pointer pada Array

ğŸ’¡ Pentingnya Pointer

Pointer sangat penting dalam pengelolaan memori dinamis, optimasi performa, dan interaksi dengan sistem operasi atau hardware pada level yang lebih rendah.
