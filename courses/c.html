Pelajaran 1: Pengenalan C Programming

üéØ Apa itu Bahasa C?

Bahasa C adalah "God's Programming Language" yang dibuat oleh Dennis Ritchie pada tahun 1969-1973. C adalah bahasa pemrograman yang:

  • Sangat powerful dan efficient
  • Menjadi basis dari bahasa modern (C++, C#, Java, PHP)
  • Digunakan untuk membuat sistem operasi (Linux, Unix, Windows)
  • Low-level namun tetap portable

üìã Struktur Program C Dasar

#include <stdio.h> int main() { printf("Hello, World!\n"); return 0; }

Penjelasan Komponen:

  • #include <stdio.h> - Menyertakan library standar input/output
  • int main() - Fungsi utama program
  • printf() - Fungsi untuk mencetak output
  • return 0; - Mengakhiri program dengan status sukses

üíª printf() - Menampilkan Output

printf("Welcome to C!\n"); // \n adalah escape character untuk baris baru // Escape characters lainnya: // \t = tab // \\ = backslash // \" = tanda kutip
üí° Tips: Setiap statement di C harus diakhiri dengan semicolon (;). Ini sangat penting!

üìù Komentar dalam C

// Ini adalah komentar satu baris /* Ini adalah komentar multi-baris yang bisa mencakup beberapa baris */

Pelajaran 2: Variabel & Tipe Data

üì¶ Apa itu Variabel?

Variabel adalah lokasi di memori tempat kita menyimpan data. Setiap variabel memiliki:

  • Nama (identifier)
  • Tipe data
  • Nilai

üî§ Tipe Data Dasar di C

// Integer - bilangan bulat int umur = 25; int suhu = -10; // Float - bilangan desimal (6-7 digit presisi) float tinggi = 175.5; float berat = 65.3; // Double - bilangan desimal (15-16 digit presisi) double phi = 3.14159265359; // Character - karakter tunggal char grade = 'A'; char huruf = 'Z'; // String - kumpulan karakter (array of char) char nama[] = "John Doe";

üìä Ukuran Tipe Data

  • int - 4 bytes (biasanya)
  • float - 4 bytes
  • double - 8 bytes
  • char - 1 byte

üîß Deklarasi dan Inisialisasi

// Deklarasi (hanya membuat variabel) int angka; // Inisialisasi (memberi nilai awal) angka = 10; // Deklarasi + Inisialisasi sekaligus int nilai = 100; // Multiple declaration int a, b, c; a = 1; b = 2; c = 3; // Multiple declaration + initialization int x = 5, y = 10, z = 15;

üìù scanf() - Mengambil Input

#include <stdio.h> int main() { int umur; float tinggi; char nama[50]; printf("Masukkan nama Anda: "); scanf("%s", nama); printf("Masukkan umur Anda: "); scanf("%d", &umur); printf("Masukkan tinggi (cm): "); scanf("%f", &tinggi); printf("\nNama: %s\n", nama); printf("Umur: %d tahun\n", umur); printf("Tinggi: %.1f cm\n", tinggi); return 0; }

Format Specifiers:

  • %d - integer
  • %f - float/double
  • %c - character
  • %s - string
  • %lf - double (untuk scanf)
‚ö†Ô∏è Perhatian: Gunakan operator & (ampersand) sebelum nama variabel di scanf, kecuali untuk string!

Pelajaran 3: Operator & Ekspresi

‚ûï Operator Aritmatika

int a = 10, b = 3; // Penjumlahan int hasil = a + b; // 13 // Pengurangan hasil = a - b; // 7 // Perkalian hasil = a * b; // 30 // Pembagian (integer division) hasil = a / b; // 3 (bukan 3.333...) // Modulus (sisa pembagian) hasil = a % b; // 1
‚ö†Ô∏è Penting: Pembagian integer di C akan membuang bagian desimalnya!
Untuk hasil desimal, gunakan float atau double.

üìä Assignment Operators

int x = 10; // Compound assignment x += 5; // x = x + 5 ‚Üí 15 x -= 3; // x = x - 3 ‚Üí 12 x *= 2; // x = x * 2 ‚Üí 24 x /= 4; // x = x / 4 ‚Üí 6 x %= 4; // x = x % 4 ‚Üí 2 // Increment & Decrement x++; // x = x + 1 ++x; // x = x + 1 x--; // x = x - 1 --x; // x = x - 1

üîç Operator Perbandingan

int a = 5, b = 10; // Hasil operator perbandingan adalah 0 (false) atau 1 (true) a == b // 0 (sama dengan?) a != b // 1 (tidak sama dengan?) a > b // 0 (lebih besar?) a < b // 1 (lebih kecil?) a >= b // 0 (lebih besar atau sama?) a <= b // 1 (lebih kecil atau sama?)

üéØ Operator Logika

int x = 5, y = 10, z = 15; // AND (&&) - kedua kondisi harus true (x < y && y < z) // 1 (true) // OR (||) - salah satu kondisi true (x > y || y < z) // 1 (true) // NOT (!) - membalik kondisi !(x == y) // 1 (true)

üìê Contoh Program Lengkap

#include <stdio.h> int main() { int angka1, angka2; printf("Masukkan angka pertama: "); scanf("%d", &angka1); printf("Masukkan angka kedua: "); scanf("%d", &angka2); printf("\nHasil Operasi:\n"); printf("%d + %d = %d\n", angka1, angka2, angka1 + angka2); printf("%d - %d = %d\n", angka1, angka2, angka1 - angka2); printf("%d * %d = %d\n", angka1, angka2, angka1 * angka2); printf("%d / %d = %d\n", angka1, angka2, angka1 / angka2); printf("%d %% %d = %d\n", angka1, angka2, angka1 % angka2); return 0; }

Pelajaran 4: Struktur Kontrol

üîÄ IF Statement

int nilai = 85; if (nilai >= 80) { printf("Nilai A - Excellent!\n"); } // IF-ELSE if (nilai >= 70) { printf("Lulus\n"); } else { printf("Tidak Lulus\n"); } // IF-ELSE IF-ELSE if (nilai >= 90) { printf("Grade: A\n"); } else if (nilai >= 80) { printf("Grade: B\n"); } else if (nilai >= 70) { printf("Grade: C\n"); } else { printf("Grade: D\n"); }

üéõÔ∏è SWITCH Statement

int hari = 3; switch (hari) { case 1: printf("Senin\n"); break; case 2: printf("Selasa\n"); break; case 3: printf("Rabu\n"); break; case 4: printf("Kamis\n"); break; case 5: printf("Jumat\n"); break; default: printf("Weekend!\n"); }

Pelajaran 5: Function & Rekursi

üéØ Apa itu Function?

Function adalah blok kode yang dapat digunakan kembali untuk melakukan tugas tertentu.

üìã Struktur Function

// Syntax: // return_type function_name(parameters) { // // body // } // Function tanpa parameter dan return void salam() { printf("Halo, selamat datang!\n"); } // Function dengan parameter void sapaNama(char nama[]) { printf("Halo, %s!\n", nama); } // Function dengan return value int tambah(int a, int b) { return a + b; } // Function dengan multiple parameters float rataRata(float a, float b, float c) { return (a + b + c) / 3.0; }

üîß Menggunakan Function

#include <stdio.h> // Function prototype (deklarasi) int kali(int x, int y); float bagi(float x, float y); int main() { int hasil1 = kali(5, 3); printf("5 x 3 = %d\n", hasil1); float hasil2 = bagi(10.0, 3.0); printf("10 / 3 = %.2f\n", hasil2); return 0; } // Function definition (implementasi) int kali(int x, int y) { return x * y; } float bagi(float x, float y) { if (y != 0) { return x / y; } else { printf("Error: Pembagi tidak boleh 0!\n"); return 0; } }

üéÆ Rekursi

Rekursi adalah teknik di mana sebuah function memanggil dirinya sendiri untuk menyelesaikan tugas. Rekursi sangat berguna untuk masalah yang dapat dipecahkan dengan membagi masalah besar menjadi masalah yang lebih kecil.

Contoh Rekursi: Faktorial

#include <stdio.h> // Fungsi rekursif untuk menghitung faktorial int faktorial(int n) { if (n == 0) { return 1; // Basis: 0! = 1 } else { return n * faktorial(n - 1); // Rekursif: n! = n * (n-1)! } } int main() { int angka = 5; printf("Faktorial dari %d adalah %d\n", angka, faktorial(angka)); return 0; }
üí° Tips: Setiap rekursi membutuhkan kondisi dasar (base case) untuk menghentikan pemanggilan berulang.

üîß Fungsi dalam Program C Lengkap

#include <stdio.h> // Deklarasi fungsi int tambah(int a, int b); int main() { int hasil = tambah(5, 10); printf("5 + 10 = %d\n", hasil); return 0; } // Definisi fungsi int tambah(int a, int b) { return a + b; }

Pelajaran 6: Array & Pointer

üìö Apa itu Array?

Array adalah kumpulan variabel yang memiliki tipe data yang sama dan disusun dalam urutan tertentu. Array sangat berguna untuk menyimpan banyak nilai dalam satu nama variabel.

üìã Deklarasi dan Akses Array

#include <stdio.h> int main() { // Deklarasi array dengan 5 elemen int angka[5] = {10, 20, 30, 40, 50}; // Mengakses elemen array printf("Elemen pertama: %d\n", angka[0]); // 10 printf("Elemen kedua: %d\n", angka[1]); // 20 return 0; }

üìä Array Multidimensi

#include <stdio.h> int main() { // Deklarasi array 2D (matrix) int matriks[2][3] = {{1, 2, 3}, {4, 5, 6}}; // Mengakses elemen array 2D printf("Matriks[0][1] = %d\n", matriks[0][1]); // 2 printf("Matriks[1][2] = %d\n", matriks[1][2]); // 6 return 0; }

üîç Apa itu Pointer?

Pointer adalah variabel yang menyimpan alamat memori dari variabel lain. Pointer digunakan untuk mengakses dan memanipulasi data di lokasi memori tertentu.

üìã Deklarasi dan Penggunaan Pointer

#include <stdio.h> int main() { int angka = 10; int *ptr = &angka; // Pointer ptr menyimpan alamat dari angka // Mengakses nilai melalui pointer printf("Nilai angka: %d\n", *ptr); // Output: 10 printf("Alamat angka: %p\n", ptr); // Output: alamat memori dari angka return 0; }

üìä Pointer pada Array

#include <stdio.h> int main() { int angka[3] = {10, 20, 30}; int *ptr = angka; // Pointer ke array // Mengakses elemen array menggunakan pointer printf("Elemen pertama: %d\n", *(ptr + 0)); printf("Elemen kedua: %d\n", *(ptr + 1)); return 0; }

üí° Pentingnya Pointer

Pointer sangat penting dalam pengelolaan memori dinamis, optimasi performa, dan interaksi dengan sistem operasi atau hardware pada level yang lebih rendah.
