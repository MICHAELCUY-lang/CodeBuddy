Pelajaran 1: Pengenalan Responsive Web Design

Responsive Web Design adalah pendekatan desain web yang membuat halaman web dapat beradaptasi dengan berbagai ukuran layar dan perangkat.

Mengapa Responsive Design Penting?

  • Penggunaan mobile device yang meningkat
  • Beragam ukuran layar (smartphone, tablet, desktop)
  • SEO benefit - Google prioritas mobile-friendly sites
  • User experience yang lebih baik
  • Maintenance yang lebih mudah (satu website untuk semua device)

Prinsip Responsive Design

/* 1. Fluid Grid System */
.container {
    width: 100%;
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 20px;
}

.row {
    display: flex;
    flex-wrap: wrap;
    margin: 0 -15px;
}

.col {
    flex: 1;
    padding: 0 15px;
}

/* 2. Flexible Images */
img {
    max-width: 100%;
    height: auto;
}

/* 3. Media Queries */
@media (max-width: 768px) {
    .col {
        flex: 100%;
    }
}

Breakpoints Umum

/* Extra Small devices (phones, 600px and down) */
@media only screen and (max-width: 600px) {...}

/* Small devices (portrait tablets and large phones, 600px and up) */
@media only screen and (min-width: 600px) {...}

/* Medium devices (landscape tablets, 768px and up) */
@media only screen and (min-width: 768px) {...}

/* Large devices (laptops/desktops, 992px and up) */
@media only screen and (min-width: 992px) {...}

/* Extra large devices (large laptops and desktops, 1200px and up) */
@media only screen and (min-width: 1200px) {...}

Fixed vs Responsive Layout

/* Fixed Layout (tidak responsive) */
.fixed-container {
    width: 960px;
    margin: 0 auto;
}

.fixed-sidebar {
    width: 300px;
    float: left;
}

.fixed-content {
    width: 660px;
    float: right;
}

/* Responsive Layout */
.responsive-container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 20px;
}

.responsive-layout {
    display: grid;
    grid-template-columns: 1fr 3fr;
    gap: 20px;
}

@media (max-width: 768px) {
    .responsive-layout {
        grid-template-columns: 1fr;
    }
}

Pelajaran 2: Viewport dan Meta Tag

Viewport meta tag adalah elemen penting untuk membuat website responsive di perangkat mobile.

Viewport Meta Tag

<!-- Basic viewport meta tag -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Dengan kontrol zoom -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">

<!-- Untuk PWA -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover">

Viewport Properties

/* width: mengatur lebar viewport */
width=device-width  /* Menggunakan lebar device */
width=320          /* Fixed width dalam pixels */

/* height: mengatur tinggi viewport */
height=device-height
height=480

/* initial-scale: zoom level awal */
initial-scale=1.0  /* No zoom */
initial-scale=0.5  /* 50% zoom out */
initial-scale=2.0  /* 200% zoom in */

/* minimum-scale & maximum-scale: batas zoom */
minimum-scale=0.5
maximum-scale=3.0

/* user-scalable: apakah user bisa zoom */
user-scalable=yes  /* Default, user bisa zoom */
user-scalable=no   /* User tidak bisa zoom */

CSS Viewport Units

/* Viewport Width (vw) - 1vw = 1% dari lebar viewport */
.full-width {
    width: 100vw; /* Full width of viewport */
}

.half-width {
    width: 50vw; /* Half width of viewport */
}

/* Viewport Height (vh) - 1vh = 1% dari tinggi viewport */
.full-height {
    height: 100vh; /* Full height of viewport */
}

.hero-section {
    height: 80vh; /* 80% of viewport height */
}

/* Viewport Minimum (vmin) - 1vmin = 1% dari dimensi terkecil */
.square {
    width: 50vmin;
    height: 50vmin;
}

/* Viewport Maximum (vmax) - 1vmax = 1% dari dimensi terbesar */
.responsive-text {
    font-size: 5vmax;
}

/* Practical examples */
.hero {
    height: 100vh;
    display: flex;
    align-items: center;
    justify-content: center;
}

.sidebar {
    width: 25vw;
    min-width: 250px;
    max-width: 400px;
}

Device Pixel Ratio

/* High DPI displays */
@media only screen and (-webkit-min-device-pixel-ratio: 2),
       only screen and (min--moz-device-pixel-ratio: 2),
       only screen and (-o-min-device-pixel-ratio: 2/1),
       only screen and (min-device-pixel-ratio: 2),
       only screen and (min-resolution: 192dpi),
       only screen and (min-resolution: 2dppx) {
    
    /* High resolution styles */
    .logo {
        background-image: url('logo@2x.png');
        background-size: 100px 50px;
    }
}

/* Retina display specific */
@media (-webkit-min-device-pixel-ratio: 2) {
    .retina-image {
        background-image: url('image@2x.jpg');
        background-size: contain;
    }
}

Pelajaran 3: Media Queries

Media queries adalah fitur CSS yang memungkinkan kita menerapkan style berbeda berdasarkan karakteristik device atau viewport.

Syntax Media Queries

/* Basic syntax */
@media media-type and (media-feature) {
    /* CSS rules */
}

/* Examples */
@media screen and (max-width: 768px) {
    .container {
        padding: 10px;
    }
}

@media print {
    .no-print {
        display: none;
    }
}

@media screen and (orientation: landscape) {
    .landscape-only {
        display: block;
    }
}

Media Types

/* all - untuk semua media types (default) */
@media all and (max-width: 768px) { ... }

/* screen - untuk layar komputer, tablet, smartphone */
@media screen and (max-width: 768px) { ... }

/* print - untuk printer dan print preview */
@media print {
    body {
        font-size: 12pt;
        color: black;
        background: white;
    }
    
    .no-print {
        display: none;
    }
    
    a:after {
        content: " (" attr(href) ")";
    }
}

/* speech - untuk screen readers */
@media speech {
    .visually-hidden {
        position: static;
        clip: auto;
    }
}

Media Features

/* Width dan Height */
@media (max-width: 768px) { ... }        /* Maksimal 768px */
@media (min-width: 769px) { ... }        /* Minimal 769px */
@media (width: 768px) { ... }            /* Tepat 768px */

@media (max-height: 600px) { ... }       /* Maksimal tinggi 600px */
@media (min-height: 601px) { ... }       /* Minimal tinggi 601px */

/* Orientation */
@media (orientation: portrait) { ... }    /* Tinggi > Lebar */
@media (orientation: landscape) { ... }   /* Lebar > Tinggi */

/* Aspect Ratio */
@media (aspect-ratio: 16/9) { ... }      /* Rasio 16:9 */
@media (min-aspect-ratio: 4/3) { ... }   /* Minimal rasio 4:3 */

/* Resolution */
@media (min-resolution: 2dppx) { ... }   /* High DPI displays */
@media (min-resolution: 192dpi) { ... }  /* 192 dots per inch */

/* Color */
@media (color) { ... }                   /* Device mendukung warna */
@media (min-color: 8) { ... }            /* Minimal 8-bit color */

/* Hover capability */
@media (hover: hover) { ... }            /* Device bisa hover */
@media (hover: none) { ... }             /* Touch devices */

/* Pointer accuracy */
@media (pointer: fine) { ... }           /* Mouse, trackpad */
@media (pointer: coarse) { ... }         /* Touch screens */

Logical Operators

/* AND operator */
@media screen and (min-width: 768px) and (max-width: 1024px) {
    /* Tablet landscape */
}

/* OR operator (comma) */
@media (max-width: 768px), (orientation: portrait) {
    /* Mobile OR portrait orientation */
}

/* NOT operator */
@media not screen and (max-width: 768px) {
    /* NOT mobile screens */
}

/* ONLY operator */
@media only screen and (max-width: 768px) {
    /* Hanya untuk screen, bukan print */
}

/* Complex queries */
@media screen and (min-width: 768px) and (max-width: 1024px) and (orientation: landscape) {
    /* Tablet landscape specific */
    .tablet-landscape {
        display: grid;
        grid-template-columns: 1fr 2fr;
    }
}

Breakpoint Strategy

/* Mobile First Approach */
/* Base styles untuk mobile */
.container {
    padding: 15px;
}

.grid {
    display: block;
}

.grid-item {
    margin-bottom: 20px;
}

/* Small tablets */
@media (min-width: 576px) {
    .container {
        padding: 20px;
    }
}

/* Large tablets */
@media (min-width: 768px) {
    .grid {
        display: grid;
        grid-template-columns: repeat(2, 1fr);
        gap: 20px;
    }
    
    .grid-item {
        margin-bottom: 0;
    }
}

/* Desktop */
@media (min-width: 992px) {
    .container {
        max-width: 1200px;
        margin: 0 auto;
    }
    
    .grid {
        grid-template-columns: repeat(3, 1fr);
    }
}

/* Large desktop */
@media (min-width: 1200px) {
    .grid {
        grid-template-columns: repeat(4, 1fr);
    }
}

Pelajaran 4: Flexible Layouts

Flexible layouts menggunakan unit relatif dan teknik modern CSS untuk membuat layout yang dapat beradaptasi dengan berbagai ukuran layar.

Flexbox untuk Responsive Layout

/* Basic Flexbox Layout */
.flex-container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.flex-item {
    flex: 1;
    min-width: 250px; /* Minimum width sebelum wrap */
}

/* Responsive Navigation */
.nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
}

.nav-links {
    display: flex;
    list-style: none;
    gap: 20px;
}

@media (max-width: 768px) {
    .nav {
        flex-direction: column;
    }
    
    .nav-links {
        width: 100%;
        justify-content: center;
        margin-top: 20px;
    }
}

/* Card Layout */
.card-container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
    justify-content: center;
}

.card {
    flex: 1 1 300px; /* grow shrink basis */
    max-width: 400px;
    min-width: 250px;
}

CSS Grid untuk Responsive Layout

/* Auto-fit Grid */
.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}

/* Auto-fill Grid */
.grid-auto-fill {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 15px;
}

/* Responsive Grid Areas */
.layout {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
    grid-template-columns: 250px 1fr 1fr;
    grid-template-rows: auto 1fr auto;
    min-height: 100vh;
    gap: 20px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

@media (max-width: 768px) {
    .layout {
        grid-template-areas:
            "header"
            "main"
            "sidebar"
            "footer";
        grid-template-columns: 1fr;
    }
}

/* Responsive Grid with Different Breakpoints */
.responsive-grid {
    display: grid;
    gap: 20px;
}

/* Mobile: 1 column */
@media (max-width: 576px) {
    .responsive-grid {
        grid-template-columns: 1fr;
    }
}

/* Tablet: 2 columns */
@media (min-width: 577px) and (max-width: 768px) {
    .responsive-grid {
        grid-template-columns: repeat(2, 1fr);
    }
}

/* Desktop: 3 columns */
@media (min-width: 769px) and (max-width: 1024px) {
    .responsive-grid {
        grid-template-columns: repeat(3, 1fr);
    }
}

/* Large Desktop: 4 columns */
@media (min-width: 1025px) {
    .responsive-grid {
        grid-template-columns: repeat(4, 1fr);
    }
}

Fluid Typography

/* Responsive Font Sizes */
html {
    font-size: 16px;
}

@media (max-width: 768px) {
    html {
        font-size: 14px;
    }
}

@media (min-width: 1200px) {
    html {
        font-size: 18px;
    }
}

/* Fluid Typography dengan clamp() */
.fluid-text {
    font-size: clamp(1rem, 2.5vw, 2rem);
    /* min: 1rem, preferred: 2.5vw, max: 2rem */
}

.heading {
    font-size: clamp(1.5rem, 4vw, 3rem);
    line-height: 1.2;
}

/* Responsive Line Height */
.paragraph {
    font-size: clamp(0.875rem, 1.5vw, 1.125rem);
    line-height: clamp(1.4, 1.5vw + 1rem, 1.6);
}

/* Viewport-based Typography */
.hero-title {
    font-size: 8vw;
    font-size: clamp(2rem, 8vw, 6rem);
}

.subtitle {
    font-size: 4vw;
    font-size: clamp(1rem, 4vw, 2rem);
}

Container Queries (Modern)

/* Container Queries - CSS yang lebih modern */
.card-container {
    container-type: inline-size;
    container-name: card;
}

.card {
    padding: 1rem;
    background: white;
    border-radius: 8px;
}

/* Query berdasarkan ukuran container, bukan viewport */
@container card (min-width: 300px) {
    .card {
        display: grid;
        grid-template-columns: auto 1fr;
        gap: 1rem;
    }
}

@container card (min-width: 500px) {
    .card {
        grid-template-columns: auto 1fr auto;
        padding: 2rem;
    }
}

/* Fallback untuk browser yang belum support */
@supports not (container-type: inline-size) {
    @media (min-width: 300px) {
        .card {
            display: grid;
            grid-template-columns: auto 1fr;
            gap: 1rem;
        }
    }
}

Pelajaran 5: Responsive Images

Responsive images memastikan gambar dapat beradaptasi dengan berbagai ukuran layar dan resolusi untuk performa dan tampilan yang optimal.

Basic Responsive Images

/* Basic responsive image */
img {
    max-width: 100%;
    height: auto;
}

/* Responsive image dengan object-fit */
.responsive-img {
    width: 100%;
    height: 200px;
    object-fit: cover; /* cover, contain, fill, scale-down, none */
    object-position: center;
}

/* Different object-fit values */
.cover-img {
    object-fit: cover; /* Crop untuk fill container */
}

.contain-img {
    object-fit: contain; /* Fit seluruh gambar dalam container */
}

.fill-img {
    object-fit: fill; /* Stretch untuk fill container */
}

/* Responsive background images */
.hero-bg {
    background-image: url('hero.jpg');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
    height: 50vh;
    min-height: 300px;
}

@media (max-width: 768px) {
    .hero-bg {
        height: 40vh;
        min-height: 250px;
        background-position: center top;
    }
}

Lazy Loading

<!-- Native lazy loading -->

    src="image.jpg" 
    loading="lazy" 
    alt="Lazy loaded image">

    src="placeholder.jpg" 
    data-src="actual-image.jpg"
    class="lazy-load"

<!-- CSS untuk lazy loading -->
<style>
.lazy-load {
    opacity: 0;
    transition: opacity 0.3s;
}

.lazy-load.loaded {
    opacity: 1;
}

.lazy-load {
    background: linear-gradient(90deg, #f0f0f0 25%, #e0e0e0 50%, #f0f0f0 75%);
    background-size: 200% 100%;
    animation: loading 1.5s infinite;
}

@keyframes loading {
    0% { background-position: 200% 0; }
    100% { background-position: -200% 0; }
}
</style>

<!-- JavaScript untuk lazy loading -->
<script>
const lazyImages = document.querySelectorAll('.lazy-load');

const imageObserver = new IntersectionObserver((entries, observer) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            const img = entry.target;
            img.src = img.dataset.src;
            img.classList.add('loaded');
            observer.unobserve(img);
        }
    });
});

lazyImages.forEach(img => imageObserver.observe(img));
</script>

Pelajaran 6: Mobile-First Design

Mobile-first design adalah pendekatan yang memulai desain dari layar terkecil (mobile) kemudian berkembang ke layar yang lebih besar.

Mengapa Mobile-First?

  • Mayoritas traffic web berasal dari mobile
  • Lebih mudah menambah fitur daripada mengurangi
  • Performance yang lebih baik di mobile
  • Progressive enhancement approach
  • Google menggunakan mobile-first indexing

Mobile-First CSS Structure

/* Base styles - Mobile first (320px+) */
.container {
    width: 100%;
    padding: 0 15px;
    margin: 0 auto;
}

.navigation {
    display: flex;
    flex-direction: column;
    background: #333;
}

.nav-item {

    border-bottom: 1px solid #555;

    padding: 20px 0;

    display: block;

    margin-bottom: 20px;
    padding: 15px;

        display: grid;
