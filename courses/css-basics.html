Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web.

CSS memungkinkan Anda untuk:

  • Mengatur warna, font, dan ukuran teks
  • Mengatur layout dan posisi elemen
  • Menambahkan efek visual seperti bayangan dan animasi
  • Membuat desain yang responsif

Cara Menambahkan CSS

Ada tiga cara untuk menambahkan CSS ke HTML:

1. Inline CSS
<p style="color: blue; font-size: 18px;">Teks berwarna biru</p>
2. Internal CSS
<head>
    <style>
        p {
            color: blue;
            font-size: 18px;
        }
    </style>
</head>
3. External CSS
<head>
    <link rel="stylesheet" href="style.css">
</head>

Pelajaran 2: CSS Selector

Selector adalah cara untuk memilih elemen HTML yang ingin diberi style.

Jenis-jenis Selector

1. Element Selector

Memilih semua elemen dengan tag tertentu:

p {
    color: red;
}

h1 {
    font-size: 24px;
}

2. Class Selector

Memilih elemen dengan class tertentu:

.highlight {
    background-color: yellow;
}

.center {
    text-align: center;
}

3. ID Selector

Memilih elemen dengan ID tertentu:

#header {
    background-color: blue;
}

#footer {
    margin-top: 20px;
}

Pelajaran 3: Properti Text dan Font

Pelajari cara mengatur tampilan teks dengan berbagai properti CSS.

Properti Font

p {
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;
    font-style: italic;
}

Properti Text

h1 {
    color: #333;
    text-align: center;
    text-decoration: underline;
    text-transform: uppercase;
    line-height: 1.5;
}

Contoh Penggunaan

.title {
    font-family: 'Georgia', serif;
    font-size: 28px;
    font-weight: bold;
    color: #2c3e50;
    text-align: center;
    margin-bottom: 20px;
}

Pelajaran 4: Box Model

Box Model adalah konsep fundamental dalam CSS yang menjelaskan bagaimana elemen HTML ditampilkan sebagai kotak.

Komponen Box Model

  • Content: Isi dari elemen
  • Padding: Ruang di dalam elemen, di sekitar content
  • Border: Garis tepi elemen
  • Margin: Ruang di luar elemen

Properti Box Model

.box {
    width: 200px;
    height: 100px;
    padding: 20px;
    border: 2px solid #333;
    margin: 10px;
    background-color: #f0f0f0;
}

Shorthand Properties

/* Margin dan Padding */
margin: 10px 20px 10px 20px; /* top right bottom left */
margin: 10px 20px; /* top/bottom left/right */
margin: 10px; /* semua sisi */

/* Border */
border: 2px solid red; /* width style color */

Pelajaran 5: Layout dengan CSS

Pelajari cara mengatur layout elemen dengan properti display dan position.

Properti Display

/* Block element */
div {
    display: block;
    width: 100%;
}

/* Inline element */
span {
    display: inline;
}

/* Inline-block */
.button {
    display: inline-block;
    padding: 10px 20px;
}

Properti Position

/* Static (default) */
.normal {
    position: static;
}

/* Relative */
.relative {
    position: relative;
    top: 10px;
    left: 20px;
}

/* Absolute */
.absolute {
    position: absolute;
    top: 0;
    right: 0;
}

/* Fixed */
.fixed {
    position: fixed;
    bottom: 0;
    right: 0;
}

Float dan Clear

.left {
    float: left;
    width: 50%;
}

.right {
    float: right;
    width: 50%;
}

.clear {
    clear: both;
}
