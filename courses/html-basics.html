Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web.

HTML digunakan untuk:

  • Membuat struktur dasar halaman web
  • Menampilkan teks, gambar, dan media lainnya
  • Membuat tautan antar halaman
  • Mengorganisir konten menjadi paragraf, heading, dan lainnya

Struktur Dasar Dokumen HTML

Setiap dokumen HTML memiliki struktur dasar sebagai berikut:

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
    <meta charset="UTF-8">
</head>
<body>
    <!-- Konten halaman web -->
    <h1>Ini adalah heading</h1>
    <p>Ini adalah paragraf.</p>
</body>
</html>
Penjelasan:
  • <!DOCTYPE html>: Deklarasi tipe dokumen
  • <html>: Elemen root dari halaman HTML
  • <head>: Berisi meta-informasi tentang dokumen
  • <title>: Menentukan judul halaman
  • <body>: Berisi konten yang terlihat di browser

Pelajaran 2: Tag dan Elemen HTML

Tag HTML adalah blok pembangun dasar dari halaman web. Mereka mendefinisikan elemen-elemen yang membentuk struktur dan konten halaman.

Tag HTML Dasar

1. Heading

HTML menyediakan enam level heading, dari h1 (paling penting) hingga h6 (paling tidak penting):

<h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>

2. Paragraf dan Teks

Tag untuk menampilkan teks dan paragraf:

<p>Ini adalah paragraf.</p>
<br> <!-- Line break (baris baru) -->
<hr> <!-- Horizontal rule (garis horizontal) -->
<strong>Teks tebal</strong>
<em>Teks miring</em>
<mark>Teks yang disorot</mark>
<small>Teks kecil</small>

3. List

Tag untuk membuat daftar:

<!-- Unordered List (Daftar tidak berurutan) -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<!-- Ordered List (Daftar berurutan) -->
<ol>
    <li>Langkah pertama</li>
    <li>Langkah kedua</li>
    <li>Langkah ketiga</li>
</ol>

Pelajaran 3: Atribut HTML

Atribut HTML memberikan informasi tambahan tentang elemen dan dapat memodifikasi perilaku atau tampilan elemen tersebut.

Atribut HTML Umum

<!-- Atribut id - Mengidentifikasi elemen secara unik -->
<div id="header">Ini adalah header</div>

<!-- Atribut class - Mengklasifikasikan elemen untuk styling -->
<p class="highlight">Teks yang disorot</p>

<!-- Atribut style - Menambahkan inline CSS -->
<p style="color: blue; font-size: 18px;">Teks berwarna biru</p>

<!-- Atribut title - Menambahkan tooltip -->
<p title="Ini adalah tooltip">Arahkan kursor ke teks ini</p>

Atribut untuk Link dan Gambar

<!-- Atribut untuk link (a) -->
<a href="https://www.example.com" target="_blank" rel="noopener">Kunjungi Example.com</a>

<!-- Atribut untuk gambar (img) -->
<img src="gambar.jpg" alt="Deskripsi gambar" width="300" height="200">

Contoh Penggunaan

.title {
    font-family: 'Georgia', serif;
    font-size: 28px;
    font-weight: bold;
    color: #2c3e50;
    text-align: center;
    margin-bottom: 20px;
}

Pelajaran 4: Semantic HTML

Semantic HTML menggunakan tag yang menggambarkan makna atau tujuan dari konten, bukan hanya tampilan. Ini membantu mesin pencari, pembaca layar, dan pengembang memahami struktur halaman.

Tag Semantic untuk Struktur Halaman

<!-- Struktur halaman dengan tag semantic -->
<header>
    <h1>Judul Website</h1>
    <nav>
        <ul>
            <li><a href="#">Beranda</a></li>
            <li><a href="#">Tentang</a></li>
            <li><a href="#">Kontak</a></li>
        </ul>
    </nav>
</header>

<main>
    <section>
        <h2>Bagian Utama</h2>
        <article>
            <h3>Artikel 1</h3>
            <p>Konten artikel...</p>
        </article>
    </section>
    
    <aside>
        <h3>Sidebar</h3>
        <p>Konten tambahan...</p>
    </aside>
</main>

<footer>
    <p>Hak Cipta © 2024</p>
</footer>

Keuntungan Menggunakan Tag Semantic

  • Meningkatkan aksesibilitas untuk pengguna dengan disabilitas
  • Memperbaiki SEO (Search Engine Optimization)
  • Membuat kode lebih mudah dibaca dan dipelihara
  • Memberikan struktur yang jelas pada dokumen

Pelajaran 5: Formulir dan Tabel

HTML menyediakan tag khusus untuk membuat formulir dan tabel yang interaktif.

Formulir HTML

<form action="/submit-form" method="post">
    <label for="nama">Nama:</label>
    <input type="text" id="nama" name="nama" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="pesan">Pesan:</label>
    <textarea id="pesan" name="pesan" rows="4" cols="50"></textarea>
    
    <label for="kategori">Kategori:</label>
    <select id="kategori" name="kategori">
        <option value="umum">Umum</option>
        <option value="teknis">Teknis</option>
        <option value="bisnis">Bisnis</option>
    </select>
    
    <fieldset>
        <legend>Jenis Kelamin:</legend>
        <input type="radio" id="pria" name="gender" value="pria">
        <label for="pria">Pria</label>
        <input type="radio" id="wanita" name="gender" value="wanita">
        <label for="wanita">Wanita</label>
    </fieldset>
    
    <input type="submit" value="Kirim">
    <input type="reset" value="Reset">
</form>

Tabel HTML

<table border="1">
    <caption>Jadwal Kelas</caption>
    <thead>
        <tr>
            <th>Hari</th>
            <th>Mata Pelajaran</th>
            <th>Waktu</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Senin</td>
            <td>Matematika</td>
            <td>08:00 - 10:00</td>
        </tr>
        <tr>
            <td>Selasa</td>
            <td>Bahasa Inggris</td>
            <td>10:30 - 12:30</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">Total: 2 mata pelajaran</td>
        </tr>
    </tfoot>
</table>
