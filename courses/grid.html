Pelajaran 1: Pengenalan CSS Grid

CSS Grid adalah sistem layout 2D yang memungkinkan kita mengatur elemen dalam baris dan kolom secara bersamaan. Grid memberikan kontrol yang lebih powerful dibanding Flexbox untuk layout kompleks.

Grid vs Flexbox

  • Grid: Layout 2D (baris dan kolom)
  • Flexbox: Layout 1D (satu arah)
  • Grid: Cocok untuk layout halaman utama
  • Flexbox: Cocok untuk komponen kecil
  • Grid: Item bisa overlap
  • Flexbox: Item tidak bisa overlap

Basic Grid Setup

/* Membuat grid container */
.grid-container {
    display: grid;
}

/* Atau inline-grid */
.inline-grid {
    display: inline-grid;
}

/* Basic 3x3 grid */
.basic-grid {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-template-rows: 100px 100px 100px;
    gap: 10px;
    background-color: #f0f0f0;
    padding: 20px;
}

.grid-item {
    background-color: #4CAF50;
    color: white;
    padding: 20px;
    text-align: center;
    border-radius: 4px;
}

Grid Terminology

/* Grid Container - parent element dengan display: grid */
.grid-container {
    display: grid;
    /* Grid Lines - garis yang membagi grid */
    /* Grid Tracks - space antara dua grid lines */
    /* Grid Cells - intersection antara row dan column */
    /* Grid Areas - rectangular area yang terdiri dari satu atau lebih cells */
}

/* Grid Items - child elements dari grid container */
.grid-item {
    /* Setiap child otomatis menjadi grid item */
}

/* Contoh dengan komentar */
.example-grid {
    display: grid;
    
    /* Defines 3 columns */
    grid-template-columns: 200px 1fr 100px;
    
    /* Defines 2 rows */
    grid-template-rows: 80px 1fr;
    
    /* Gap between grid items */
    gap: 20px;
    
    /* Grid container properties */
    height: 400px;
    border: 2px solid #333;
}

Grid Units

/* fr (fraction) unit - distribusi space yang tersisa */
.fr-example {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr; /* 1:2:1 ratio */
}

/* Fixed units */
.fixed-units {
    display: grid;
    grid-template-columns: 200px 300px 150px; /* Fixed pixels */
    grid-template-rows: 100px 200px; /* Fixed heights */
}

/* Percentage units */
.percentage-units {
    display: grid;
    grid-template-columns: 25% 50% 25%;
    grid-template-rows: 30% 70%;
}

/* Mixed units */
.mixed-units {
    display: grid;
    grid-template-columns: 200px 1fr 100px; /* Fixed-Flexible-Fixed */
    grid-template-rows: auto 1fr auto; /* Content-Flexible-Content */
}

/* min-content, max-content, auto */
.content-units {
    display: grid;
    grid-template-columns: min-content max-content auto;
}

/* repeat() function */
.repeat-example {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* Same as: 1fr 1fr 1fr */
    grid-template-rows: repeat(2, 100px); /* Same as: 100px 100px */
}

/* repeat with mixed values */
.repeat-mixed {
    display: grid;
    grid-template-columns: 200px repeat(3, 1fr) 100px;
}

Pelajaran 2: Grid Container Properties

Grid container adalah parent element yang menggunakan display: grid. Container memiliki berbagai properties untuk mengontrol struktur dan behavior grid.

grid-template-columns & grid-template-rows

/* Explicit grid definition */
.explicit-grid {
    display: grid;
    
    /* Define 3 columns */
    grid-template-columns: 100px 200px 1fr;
    
    /* Define 2 rows */
    grid-template-rows: 80px 120px;
}

/* Using repeat() */
.repeat-grid {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    grid-template-rows: repeat(3, 100px);
}

/* Mixed repeat patterns */
.mixed-repeat {
    display: grid;
    grid-template-columns: 200px repeat(3, 1fr) 100px;
    grid-template-rows: auto repeat(2, 1fr) auto;
}

/* Named grid lines */
.named-lines {
    display: grid;
    grid-template-columns: 
        [sidebar-start] 250px 
        [sidebar-end main-start] 1fr 
        [main-end];
    grid-template-rows: 
        [header-start] 80px 
        [header-end content-start] 1fr 
        [content-end footer-start] 60px 
        [footer-end];
}

/* Using minmax() */
.minmax-grid {
    display: grid;
    grid-template-columns: 
        minmax(200px, 300px) 
        minmax(300px, 1fr) 
        minmax(100px, 200px);
    grid-template-rows: 
        minmax(80px, auto) 
        minmax(200px, 1fr);
}

/* fit-content() */
.fit-content-grid {
    display: grid;
    grid-template-columns: 
        fit-content(200px) 
        1fr 
        fit-content(150px);
}

grid-auto-columns & grid-auto-rows

/* Implicit grid sizing */
.implicit-grid {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    /* Only 2 columns defined, but we have more items */
    
    /* Size for auto-generated rows */
    grid-auto-rows: 100px;
    
    /* Size for auto-generated columns */
    grid-auto-columns: 150px;
}

/* Auto-sizing with minmax */
.auto-minmax {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-auto-rows: minmax(100px, auto);
    gap: 10px;
}

/* Different auto sizes */
.auto-sizes {
    display: grid;
    grid-template-columns: 1fr 1fr;
    grid-auto-rows: 80px;
    grid-auto-columns: 200px;
}

/* Auto with content-based sizing */
.auto-content {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-auto-rows: max-content;
    gap: 15px;
}

grid-auto-flow

/* Controls how auto-placed items flow */

/* row (default) - fill rows first */
.flow-row {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-auto-flow: row;
    gap: 10px;
}

/* column - fill columns first */
.flow-column {
    display: grid;
    grid-template-rows: repeat(3, 100px);
    grid-auto-flow: column;

    gap: 10px;
}

/* dense - fill holes in grid */
.flow-dense {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    grid-auto-flow: row dense;
    gap: 10px;
}

/* Some items with specific placement */
.flow-dense .item-large {
    grid-column: span 2;
}

/* row dense */
.flow-row-dense {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-auto-flow: row dense;
    gap: 10px;
}

/* column dense */
.flow-column-dense {
    display: grid;
    grid-template-rows: repeat(3, 100px);
    grid-auto-flow: column dense;
    grid-auto-columns: 1fr;
    gap: 10px;
}

gap (grid-gap)

/* Gap between grid items */

/* Single value - same for rows and columns */
.gap-single {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

/* Two values - row-gap column-gap */
.gap-double {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px 10px; /* 20px row gap, 10px column gap */
}

/* Individual gap properties */
.gap-individual {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    row-gap: 25px;
    column-gap: 15px;
}

/* Responsive gaps */
.gap-responsive {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 15px;
}

@media (min-width: 768px) {
    .gap-responsive {
        gap: 30px;
    }
}

/* No gap */
.no-gap {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    gap: 0;
}

/* Large gaps for spacing effect */
.large-gap {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    gap: 50px;
    padding: 50px;
}

Pelajaran 3: Grid Items Properties

Grid items adalah child elements dari grid container. Setiap item dapat ditempatkan secara spesifik dalam grid menggunakan berbagai properties.

grid-column & grid-row

/* Positioning items with line numbers */
.grid-container {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    grid-template-rows: repeat(3, 100px);
    gap: 10px;
}

/* grid-column: start / end */
.item-1 {
    grid-column: 1 / 3; /* From line 1 to line 3 (spans 2 columns) */
    grid-row: 1 / 2;
}

.item-2 {
    grid-column: 3 / 5; /* From line 3 to line 5 (spans 2 columns) */
    grid-row: 1 / 2;
}

.item-3 {
    grid-column: 1 / 5; /* Full width (spans all 4 columns) */
    grid-row: 2 / 3;
}

/* Using span keyword */
.item-span {
    grid-column: span 2; /* Span 2 columns from current position */
    grid-row: span 1;
}

/* Negative line numbers (count from end) */
.item-negative {
    grid-column: 1 / -1; /* From first line to last line */
    grid-row: 3 / -1;
}

/* Individual properties */
.item-individual {
    grid-column-start: 2;
    grid-column-end: 4;
    grid-row-start: 1;
    grid-row-end: 3;
}

Named Grid Lines

justify-self & align-self

/* Override container alignment for individual items */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 200px);
    grid-template-rows: repeat(2, 150px);
    justify-items: center; /* Default for all items */
    align-items: center;
    gap: 10px;

.justify-start {
    justify-self: start; /* Override to left align */
}

.justify-end {

.justify-center {
    justify-self: center; /* Same as container default */
}

.justify-stretch {
    justify-self: stretch; /* Fill the cell width */
}

/* align-self - vertical alignment */
.align-start {
    align-self: start; /* Override to top align */
}

.align-end {
    align-self: end; /* Override to bottom align */
}

.align-center {
    align-self: center; /* Same as container default */
}

.align-stretch {
    align-self: stretch; /* Fill the cell height */
}

/* place-self shorthand */
.place-start {
    place-self: start; /* Both align-self and justify-self to start */
}

.place-end {
    place-self: end; /* Both align-self and justify-self to end */
}

.place-center {
    place-self: center; /* Both align-self and justify-self to center */
}

.place-stretch {
    place-self: stretch; /* Both align-self and justify-self to stretch */
}

.place-mixed {
    place-self: start end; /* align-self: start, justify-self: end */
}

/* Practical example */
.card {
    place-self: center;
    width: 150px;
    height: 100px;
    background: white;
    border: 1px solid #ddd;
    border-radius: 8px;

    height: auto;
}
